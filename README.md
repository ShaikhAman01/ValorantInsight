# ValorantInsight

## Description:
ValorantInsight is a web application that provides in-depth information about Valorant agents and their abilities using the official Valorant API. The app is designed to offer players comprehensive insights into each agent, helping them understand and optimize gameplay strategies. 

## Features:
- View detailed information about Valorant agents
- Agent-specific abilities and stats
- Easy-to-use and responsive interface

## Technologies Used:
- Express.js (Backend)
- Valorant API (Data source)
- HTML, CSS, JavaScript (Frontend)

## How to Run:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ShaikhAman01/ValorantInsight
   ```

2. **Navigate into the project directory**:
   ```bash
   cd ValorantInsight
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```

5. Open `http://localhost:3000` in your browser to view the app.
